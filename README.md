# @fox-hound-tools/protobuf-grpc-json-transcoding

# Development

## 1. Install dependencies

```bash
pnpm install
```

## 2. Build

```bash
pnpm run build
```

## 3. Test

```bash
pnpm test
```

# Sample

## 1. Generate protofiles from typespec

```bash
pnpm run sample:generate
```

## 2. Run server

```bash
dotnet run --project sample/Server
```

## 3. Access to the server

http://localhost:5000
